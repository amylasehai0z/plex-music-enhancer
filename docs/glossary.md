# Glossar

## Album Context

Das normalisierte Datenmodell für ein Album. Es enthält Plex-Daten, Providerdaten, Verifikation, Stil- und Pipelineinformationen.

## Artist Context

Das normalisierte Datenmodell für einen Künstler.

## Apply

Der sichere Schreibvorgang nach Plex mit Backup, Schreiben, Reload-Verifikation und Audit.

## Audit

Ein Protokoll über eine Analyse oder einen Apply-Vorgang.

## Cache

Lokaler Zwischenspeicher für Providerdaten.

## Diff

Darstellung der Unterschiede zwischen aktuellem und neuem Text.

## DummyProvider

Lokaler AI-Testanbieter ohne Netzwerkzugriff.

## Editorial Style Engine

Komponente zur Analyse und Verbesserung stilistischer Qualität deutscher Texte.

## Fact Verification

Bewertung gesammelter Fakten nach Zuverlässigkeit.

## GPT

Sprachmodell, das aus einem Prompt Text erzeugt.

## Last.fm

Metadatenquelle für Community-Tags, Hörstatistiken und Biografien.

## MBID

MusicBrainz Identifier. Eine stabile ID für Künstler, Release oder Release Group.

## Metadata

Strukturierte Informationen über Künstler, Alben, Veröffentlichungen, Credits oder Stile.

## MusicBrainz

Offene Musikdatenbank mit stabilen IDs und strukturierten Releases.

## Pipeline

Abfolge von Schritten, die aus Plex-Daten einen geprüften Kontext und Text erzeugt.

## Plex Token

Geheimer Zugriffsschlüssel für den Plex Server.

## Prompt

Textvorlage mit Anweisungen und Variablen für das Sprachmodell.

## Prompt Version

Version einer Prompt-Vorlage. Wichtig für Reproduzierbarkeit und inkrementelle Verarbeitung.

## Provider

Eine Quelle für Metadaten, zum Beispiel MusicBrainz oder Wikipedia.

## Quality Score

Zahl von 0 bis 100 zur Bewertung eines generierten Textes.

## Release Group

MusicBrainz-Konzept für eine Veröffentlichungseinheit, die verschiedene Releases eines Albums bündelt.

## Review

Manuelle Prüfung vor einem möglichen Apply.

## SQLite

Lokale Datenbankdatei für Verarbeitungsstatus.

## Unified Diff

Standardformat zur Darstellung von Textänderungen.

## Wikipedia Extract

Kurzer Auszug aus einem Wikipedia-Artikel.
