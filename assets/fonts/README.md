# Schriften fuer PDF-Builds

Das PDF-Build-System verwendet bevorzugt lokal installierte Schriften:

- Noto Serif
- Noto Sans
- JetBrains Mono

Wenn diese Schriften nicht installiert sind, verwendet das Build-Skript TeX-Schriften als Fallback. In diesem Verzeichnis muessen keine Font-Dateien eingecheckt werden.
